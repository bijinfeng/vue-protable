# ProTable Vue — 产品需求文档

> 本项目是 [Ant Design Pro Table](https://github.com/ant-design/pro-components) 的 Vue 版本，目标是在 Vue 生态中提供同等能力的高级表格组件。

---

## 一、产品定位

封装企业级数据表格的通用逻辑，将「查询表单 + 数据请求 + 表格展示 + 工具栏操作」整合为一个组件，消除重复性的样板代码，让开发者聚焦业务逻辑。

核心设计理念：**Headless（无头）**——只输出数据和状态，不绑定任何 UI 框架，可与 Element Plus、Ant Design Vue、Naive UI 等任意 Vue UI 库结合使用。

---

## 二、核心功能

### 2.1 数据请求

- 通过 `request` 配置远程数据源，自动管理加载状态
- 支持 `postData` 对接口返回数据进行预处理
- 支持 `params` 注入额外请求参数（优先级高于表单参数）
- 支持 `debounceTime` 防抖，避免频繁触发请求
- 支持 `polling` 轮询刷新（最低间隔 2000ms），页面隐藏时自动暂停
- 支持 `revalidateOnFocus` 窗口聚焦时自动刷新
- 支持 `manualRequest` 手动触发首次请求
- 支持直接传入 `dataSource` 使用本地数据
- 支持 `defaultData` 设置初始数据
- 数据变更、加载完成、请求失败均有对应回调事件

### 2.2 查询表单

- 内置搜索表单，字段由列配置自动生成，无需额外定义
- 支持 `query`（标准展开）和 `light`（轻量悬浮）两种表单形态
- 支持折叠/展开，默认折叠，可配置默认展开状态
- 折叠时可显示隐藏字段数量提示
- 支持自定义查询按钮、重置按钮、提交按钮的文案
- 支持 `beforeSearchSubmit` 在提交前对参数进行转换
- 支持响应式列数配置（不同断点展示不同列数）
- 支持自定义操作区渲染
- 支持完全自定义整个搜索表单区域

### 2.3 列配置

**基础展示**

- 列标题支持 tooltip 提示
- 支持 `ellipsis` 超长内容自动截断并显示 tooltip
- 支持 `copyable` 一键复制单元格内容
- 支持 `columnEmptyText` 配置空值的占位展示（默认 `-`）
- 支持 `dateFormatter` 统一控制日期格式化方式

**值类型（valueType）**

- 通过 `valueType` 声明字段类型，自动匹配对应的展示和编辑控件
- 内置常用类型：文本、数字、日期、日期范围、时间、金额、进度、评分、开关、图片、链接、颜色、代码、JSON 等

**枚举映射（valueEnum）**

- 通过 `valueEnum` 配置枚举值与展示标签的映射关系
- 支持配置徽标状态色（成功、失败、警告、处理中等）
- 表头筛选菜单可自动从 `valueEnum` 生成

**排序与筛选**

- 支持列排序（本地排序和服务端排序）
- 支持表头筛选菜单（本地过滤和服务端过滤）

**列显隐控制**

- `hideInTable`：在表格中隐藏，仅在查询表单中显示
- `hideInForm`：在表单中隐藏，仅在表格中显示
- `hideInSearch`：从查询表单中排除
- `hideInSetting`：从列设置面板中排除
- `hideInDescriptions`：在描述列表中隐藏

**查询表单定制**

- 通过 `order` 控制字段在查询表单中的排列顺序
- 支持 `search.transform` 对查询参数进行字段名转换
- 支持 `initialValue` 设置查询表单默认值
- 支持 `colSize` 控制字段占用的表单列数
- 支持 `formItemProps` 配置校验规则等表单项属性
- 支持 `fieldProps` 定制输入控件的属性

**自定义渲染**

- `render`：自定义单元格内容渲染
- `renderText`：仅用于纯文本场景的简化渲染
- `renderFormItem`：自定义查询表单的输入控件

### 2.4 工具栏

- 支持 `headerTitle` 配置表格左上角标题
- 内置工具按钮：刷新、密度切换、列设置、全屏
- 支持通过 `options` 控制各工具按钮的显示/隐藏
- 支持 `toolBarRender` 在工具栏右侧添加自定义按钮
- 支持 `optionsRender` 完全自定义工具区渲染
- 工具栏支持标签页（tabs）、多行（multipleLine）等布局模式
- 支持配置副标题、tooltip 说明、搜索框嵌入工具栏

### 2.5 列设置

- 提供列设置面板，用户可交互式控制列的显示/隐藏
- 支持列顺序拖拽调整（固定列左/右）
- 支持通过 `columnsState` 持久化列的显隐状态（localStorage / sessionStorage）
- 支持受控模式，通过外部状态管理列配置

### 2.6 分页

- 内置分页，与 `request` 联动，自动传入 `current` 和 `pageSize`
- 支持配置分页器的各项参数，或传入 `false` 关闭分页
- 支持通过 `actionRef` 手动设置分页信息

### 2.7 行选择与批量操作

- 支持 `rowSelection` 配置行多选
- 选中后在表格顶部显示提示条，展示选中数量
- 支持自定义提示条左侧内容和右侧操作按钮
- 提供 `clearSelected` 方法清空选中状态
- 支持 `alwaysShowAlert` 始终显示提示条

### 2.8 可编辑表格

- 支持行内编辑，可配置单行编辑或多行同时编辑
- 支持通过 `RecordCreator` 新增行（支持顶部或底部插入）
- 支持 `editable` 列级配置控制某列是否可编辑
- 支持 `readonly` 只读模式
- 可连接外部 Form 实现表单联动
- 支持 `ignoreRules` 跳过校验规则

### 2.9 命令式操作（ActionRef）

通过 `actionRef` 可在组件外部触发以下操作：

| 方法 | 说明 |
|------|------|
| `reload()` | 刷新数据（可选是否重置页码） |
| `reloadAndRest()` | 刷新并重置表单和页码 |
| `reset()` | 重置所有状态至默认值 |
| `clearSelected()` | 清空行选中状态 |
| `startEditable(rowKey)` | 开始编辑指定行 |
| `cancelEditable(rowKey)` | 取消编辑指定行 |
| `scrollTo(key)` | 滚动到指定位置 |
| `fullScreen()` | 切换全屏状态 |
| `setPageInfo()` | 手动设置分页信息 |

### 2.10 布局与外观

- 支持 `ghost` 模式去除外层卡片内边距
- 支持 `defaultSize` 设置表格默认密度（默认/中/紧凑）
- 支持 `cardBordered` 控制卡片边框
- 支持 `tableRender` 完全接管表格区域渲染
- 支持 `tableExtraRender` 在搜索区与表格之间插入自定义内容
- 支持 `tableViewRender` 替换表格视图（如切换为列表视图）
- 内置错误边界，异常不会导致整个页面崩溃（可关闭）

## 三、使用场景

| 场景 | 支持能力 |
|------|----------|
| 带查询条件的数据列表 | 查询表单 + request 自动联动 |
| 需要批量操作的管理表格 | 行选择 + 批量操作提示条 |
| 可在线编辑的数据表格 | 可编辑表格 + 行内表单校验 |
| 需要个性化列配置的表格 | 列设置 + 状态持久化 |
| 实时监控数据大屏 | 轮询刷新 + 全屏模式 |
| 复杂筛选的报表 | 多值类型 + 枚举映射 + 服务端排序筛选 |

---

## 四、不在范围内

- 图表组件
- 树形穿梭框
- 复杂甘特图等可视化
- 移动端适配（以桌面端为主）
